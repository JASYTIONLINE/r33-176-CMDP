## 1. PURPOSE

This SOP establishes procedures for tool room operations, accountability, and security within [UNIT NAME] in accordance with AR 710-2, CTA 50-970, and DA Pam 750-1.

---

## 2. APPLICABILITY

This SOP applies to all personnel assigned or attached to [UNIT NAME] who access the tool room or sign out tools and equipment.

---

## 3. REFERENCES

- AR 710-2, Supply Policy Below the National Level
- CTA 50-970, Expendable/Durable Items
- DA Pam 750-1, Commanders' Maintenance Handbook
- Unit Property Book and Hand Receipts

---

## 4. RESPONSIBILITIES

### 4.1 Commander
- Responsible for tool accountability
- Appoints Tool Room Custodian in writing
- Ensures tool room security

### 4.2 Tool Room Custodian
- Maintains tool accountability
- Conducts inventories
- Manages tool issue/turn-in

### 4.3 All Personnel
- Sign out tools properly
- Return tools in serviceable condition
- Report damaged or missing tools

---

## 5. PROCEDURES

### 5.1 Tool Room Access
a. Only authorized personnel access tool room  
b. Tool room secured when unattended  
c. Access roster posted and current  
d. Key control maintained IAW SOP

### 5.2 Tool Issue
a. **Individual Sign-Out:**
   - Present valid ID
   - Complete tool sign-out log
   - Inspect tool for serviceability
   - Return tools same day unless authorized overnight

b. **Section/Team Sign-Out:**
   - Supervisor authorizes sign-out
   - Complete hand receipt if extended period
   - Conduct joint inventory at issue and turn-in

### 5.3 Tool Turn-In
a. Clean tools before return  
b. Tool Room Custodian inspects tools  
c. Report damaged or unserviceable tools  
d. Document turn-in in log  
e. Initiate statement of charges/FLIPL for lost/damaged tools

### 5.4 Tool Inventories
a. **Daily:**
   - Verify all signed-out tools returned
   - Secure tool room at end of duty day

b. **Monthly:**
   - Conduct 100% inventory of tool room
   - Verify against hand receipt/property book
   - Document results

c. **Change of Custodian:**
   - Conduct joint inventory
   - Document on DA Form 2062
   - Both custodians sign inventory

### 5.5 Tool Maintenance
a. Inspect tools for serviceability regularly  
b. Remove unserviceable tools from service  
c. Coordinate replacement of unserviceable tools  
d. Calibrated tools maintained IAW calibration schedule

### 5.6 Tool Set Management
a. Maintain tool sets IAW SC/CTA authorization  
b. Conduct monthly inventory of tool sets  
c. Shortage annexes maintained and current  
d. Replace missing components promptly

---

## 6. TOOL ROOM SECURITY

- Tool room secured with approved lock
- Key control IAW AR 190-11
- Limited access roster
- Inventory conducted after any security incident

---

## 7. RECORDS AND REPORTS

- DA Form 2062 (Hand Receipt)
- Tool sign-out log
- Inventory records
- Shortage annexes
- FLIPL/Statement of Charges (as required)

---

## 8. SPECIAL TOOLS

- Calibrated tools tracked separately
- Calibration dates verified before issue
- Special tools signed out to qualified personnel only
- Document special tool usage
